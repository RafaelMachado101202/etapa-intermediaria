Sistema de incentivo ao descarte consciente com gamificação, agora totalmente web e integrado com serviços de localização.

 Sobre o Projeto
O EcoPonto Inteligente evoluiu! Na versão 2.0, deixamos o terminal para oferecer uma interface web moderna. O objetivo continua sendo estimular hábitos sustentáveis, mas agora com a facilidade de uso do navegador e inteligência de dados geográficos.

 Novidades da v2.0
Integração com API Externa: Utilização da API ViaCEP para preenchimento automático de localidade (Cidade/UF) a partir do CEP.
Interface Web (GUI): Design responsivo criado com HTML5 e CSS3.
Deploy Automatizado: Hospedagem pública via GitHub Pages.
Persistência de Dados: Uso de LocalStorage para manter seus pontos e histórico mesmo após fechar o navegador.

 Funcionalidades
 -Registro de descarte por tipo de material.
 -Busca Automática de Endereço: Identifica onde o descarte foi realizado via CEP.
 -Cálculo de EcoPontos em tempo real.
 -Sistema de níveis progressivo.
 -Histórico dos últimos descartes com registro de localidade.

 Tecnologias Utilizadas
-HTML5 & CSS3 (Interface e Estilização)
-JavaScript (ES6+) (Lógica e Manipulação de DOM)
-API ViaCEP (Serviço REST externo para busca de CEP)
-Git & GitHub (Controle de versão e Gestão de Issues)
-GitHub Pages (Deploy e Hospedagem)
